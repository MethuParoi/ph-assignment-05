What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the structure of a webpage as a tree of objects. It allows developers to access, modify, and manipulate HTML and XML content dynamically using scripting languages like JavaScript, enabling interactive and responsive web pages.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like `document.getElementById('id')` for selecting by ID, `document.querySelector('.class')` for selecting by class or tag, or `document.querySelectorAll('selector')` to select multiple elements matching a CSS selector.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events from its child elements. It's useful because it improves performance by reducing the number of event listeners and works dynamically for elements added later.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `setAttribute()`, `getAttribute()`, or directly via properties like `element.id`. For styles, use `element.style` to modify inline styles (e.g., `element.style.backgroundColor = 'red'`) or modify classes using `classList.add()`, `classList.remove()`, or `classList.toggle()`.
